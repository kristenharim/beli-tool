🍽️ Beli Staging

Turn your Google Maps saved lists + geotagged photos into a phone worklist for Beli.

Download for macOS

Setup (once)

  1. Unzip and drag Beli Staging.app to Applications.
  2. First launch: right-click → Open (it's unsigned, so a normal double-click is blocked once).
  3. Create ~/beli-tool/config.toml with your Google Places API key:
    google_places_api_key = "YOUR_KEY"
  4. Drop a Google Takeout “Maps (your places)” export into ~/beli-tool/inbox/.

Each run

  1. Double-click the app. It reads your photos + saved places (~a minute).
  2. A dialog shows a http://…:8000 URL — open it on your phone.
  3. Rank each place and tap it into Beli. Quit the app when done.

Open source at github.com/kristenharim/beli-tool. Unsigned build — macOS will ask you to confirm on first open.
